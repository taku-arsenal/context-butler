# 09 Measurement Criteria

**プロジェクト名**: Context Butler / 説明補足AI（Explain Bot）

---

## 評価方針

MVP の評価は、**審査員・デモ参加者に伝わる体験価値を重視した定性評価を中心**にします。あわせて、事前に用意したテストデータごとの期待補足ポイントをどれだけ満たせたかを定量評価します。

> **注意**: 手動修正回数は評価者や文章の好みに左右されやすいため、主要指標にしません。

---

## 1. 定性評価（主要指標）

| 指標 | 評価方法 | 目標 |
|------|---------|------|
| 補足文を読んで元投稿の意図を理解できるか | デモ参加者・審査員が補足文を読んで元投稿の意図・背景・次アクションを説明できるか確認する | 大半の参加者が意図・背景・次アクションを説明できる |
| 追加質問が減る感覚があるか | 「この補足があれば追加質問が減るか」を定性評価する | 審査員に業務利用イメージが伝わる |
| テーマ適合の伝達 | 「説明責任をAIに外注し、人をダメにする」狙いが説明できるか確認する | 「人をダメにする」テーマとの接続を審査員に説明できる |
| 事実・推測の分離 | 不明点を断定せず、根拠のある情報と分けているか確認する | 重大な根拠なし断定がない |
| 補足文の自然さ | Slack スレッドの返信として自然な文体・長さになっているか確認する | 読んで違和感がない |

---

## 2. 定量評価

### 2.1 想定補足ポイント充足率（主要定量指標）

事前に用意したテストデータに対して、想定補足ポイントをどれだけ満たせたかを採点します。

**評価方法**:

1. テストデータ（5〜10 パターンの Slack 投稿）を事前に用意する
2. 各テストデータに「想定補足ポイント」を事前定義する（例: 背景の言及・用語説明・次アクション明示・不明点の明記 など）
3. Context Butler が生成した補足文が、各想定補足ポイントを満たしているかを採点する
4. 充足率 = 満たしたポイント数 / 全想定補足ポイント数 × 100

**目標**: 80% 以上

**テストデータの例**:

```
投稿: 「例の認証の件、来週から切り替える方向で。影響あるところだけ確認お願いします。」

想定補足ポイント:
  [ ] 「例の認証」が何かを説明している（背景）
  [ ] なぜ切り替えるかを説明している（判断理由）
  [ ] 誰が何を確認すべきかを説明している（次アクション）
  [ ] 不明な点（本番適用日・ロールバック手順）を「不明」と明記している
  [ ] 事実と推測を分けて書いている
```

### 2.2 システム指標

| 指標 | 目標値 | 計測方法 |
|------|--------|---------|
| Slack Interactivity への応答時間 | 3 秒以内 | Lambda Ack の処理時間を CloudWatch で計測 |
| 補足文生成完了時間 | 30 秒以内（目標） | Worker Lambda の処理時間を CloudWatch で計測 |
| job 処理成功率 | 95% 以上（目標） | DynamoDB の POSTED / FAILED 件数から算出 |
| デモ中の補足生成完了 | デモ中に補足文が生成される | デモ本番で確認 |

---

## 3. 評価しない指標

| 指標 | 除外理由 |
|------|---------|
| 手動修正回数 | 評価者や文章の好みに左右されやすく、客観的な比較が難しい |
| 補足文の文字数 | 長さより内容の充実度を重視するため |
| ユーザー満足度スコア | MVP のデモ規模では統計的に有意なサンプル数を確保できないため |

---

## 4. デモ完成度の定義

### 予選デモの最低ライン（合格条件）

1. Slack の三点リーダから起動できる
2. 数秒後にスレッドに補足文が返信される
3. 補足文に背景・前提・次アクションが含まれる
4. Slack スレッドと社内ナレッジ KB を使った文脈取得が動く
5. AgentCore Runtime + Strands Orchestrator で 4 Agent ステージの処理ログを見せられる

### 予選デモの理想ライン

1. 上記に加えて
2. 省略抽出 Agent の構造化 JSON 出力（何が省略されているかの分析）を見せられる
3. Drive・GitHub からの文脈取得を見せられる（または説明できる）
4. 想定補足ポイント充足率 80% 以上を達成している
