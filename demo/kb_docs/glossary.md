# プロジェクト Alpha 用語集

## AWS 関連

### Cognito
AWS の認証サービス。ユーザー管理・認証・認可を提供する。
ユーザープール（認証）とアイデンティティプール（認可）の 2 種類がある。
本プロジェクトでは独自認証の代替として採用予定。

### Bedrock
AWS の生成 AI サービス。Claude・Titan・Nova などの LLM を API 経由で利用できる。

### Lambda
AWS のサーバーレスコンピューティングサービス。コードをサーバー管理なしで実行できる。

## プロジェクト固有の用語

### 独自認証
現在使用している社内開発の認証システム。
ユーザー情報を独自 DB で管理し、JWT トークンを発行する。
運用負荷が高いため、Cognito への移行が検討されている。

### Issue #123
GitHub の Issue。認証方式を Cognito に移行する提案が記載されている。
担当: 開発チーム全員
ステータス: In Progress

### プロジェクト Alpha
社内ポータルの刷新プロジェクト。
認証基盤の移行・UI 改善・パフォーマンス改善が主な目標。
